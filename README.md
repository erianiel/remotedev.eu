# Remote dev jobs in Europe

RemoteDev.eu is a lightweight job board for finding remote developer jobs available across Europe.

Live site: https://remotedev.eu

This project is a frontend application built to browse remote engineering roles in Europe through a fast, table-based interface. The UI keeps the experience minimal and practical: no account creation, no complex navigation, and no unnecessary distractions.

## ✨ Features

- Browse remote developer jobs in a clean table
- Filter by country and company
- Sort jobs by date
- Change page size
- Open the original job post in a new tab
- Mobile-friendly layout

## 🧱 Tech stack
